## What is Hacktoberfest?

[Hacktoberfest](https://hacktoberfest.com/) is a month-long celebration of Open Source that encourages people to contribute to open source throughout October. This year marks the 12th edition of Hacktoberfest. Every year, thousands of contributors participate in this festival to contribute to the projects they love and use, learn and enhance their skills to advance their careers, and get mentorship and feedback from industry experts. For organizations that believe in giving back to the community, Hacktoberfest is a fantastic platform.

## Interledger and Open Source üåê

The Interledger Foundation aims to enhance access to digital financial services by advocating for adopting an open-source currency network powered by the Interledger Protocol (ILP). Moreover, it actively collaborates with historically marginalized groups to fulfill this mission. These core values inspire us at the [Interledger Foundation to participate in Hacktoberfest](/hacktoberfest).

This year marks the fourth edition of the Interledger Foundation participating in Hacktoberfest.

## A few repositories that are participating in Hacktoberfest

### Rafiki

Rafiki is open-source software that allows an Account Servicing Entity to enable Interledger functionality on its users' accounts.

This includes:


- sending and receiving payments (via SPSP and Open Payments)
- allowing third-party access to initiate payments and view transaction data (via Open Payments)

**Repo link** -[ https://github.com/interledger/rafiki](https://github.com/interledger/rafiki)

**Contribution guide** - [https://github.com/interledger/rafiki/blob/main/.github/contributing.md](https://github.com/interledger/rafiki/blob/main/.github/contributing.md)

### Testnet

Testnet is an open-source test application for Rafiki. In other words, a Rafiki Playground. It wants to use all of the functionalities of Rafiki and put its advantages to the real test.

Testnet is made up (or will be made up soon) of several components, including a wallet application, a bank application, and an e-commerce application.

**Repo link** - [https://github.com/interledger/testnet](https://github.com/interledger/testnet)

**Contribution guide** - [https://github.com/interledger/testnet/blob/main/.github/contributing.md](https://github.com/interledger/testnet/blob/main/.github/contributing.md)

### Helm charts

This is a Helm Charts repository for Interledger Kubernetes.

**Repo link** - [https://github.com/interledger/helm-charts](https://github.com/interledger/helm-charts)

and many more on [Interledger repositories.](https://github.com/interledger)

### Web Monetization

Web Monetization browser extension is an open-source implementation of the Web Monetization draft specification. It is built with React and TypeScript.

Learn more at the links below.


- Web Monetization extension Repo link - [https://github.com/interledger/web-monetization-extension](https://github.com/interledger/web-monetization-extension)
- Web Monetization specification - [https://webmonetization.org/specification/](https://webmonetization.org/specification/)
