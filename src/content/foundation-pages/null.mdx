## Program Overview

The Interledger Foundation’s Grant for the Web program has awarded $400,000 in total to support content publishers and publishing platforms since 2025. Grantees are exploring the viability of a revenue model between ad-supported access and paywalls using Web Monetization.

## Web Monetization

Web Monetization is a technology that allows content creators, publishers, and owners to earn revenue directly from their audiences through small, automatic payments based on real engagement. It is designed as an open web standard and can complement existing monetization models, rather than replacing them. The long-term vision is a web where monetization is built into browsers, empowering content owners to thrive in a fair digital economy by removing barriers between users and the content they love.

Web Monetization’s development is led by the Interledger Foundation and financial service providers supporting the Open Payments specification within a World Wide Web Consortium (W3C) technology standards incubator group.

## Featured grantees

- The Social Web Foundation is researching sustainable revenue models for digital publishers, operating and governance models for community-run platforms, and ways to use Web Monetization with multimedia content.
- A New Social operates Bridgy Fed to provide interoperability between decentralized social network protocols (ActivityPub and AT Protocol).
- Castopod is the leading open source podcast hosting platform used by over 1,000 podcasts.
- Conffab is an online learning platform featuring content from experts curated by leading conference organizers.
- Smashing Magazine publishes reliable, useful, and practical articles about web design and development written by industry experts.
- WriteFreely is an open source publishing platform used by over 550,000 writers.
